# AI API VS Code Assignment

## Project Overview
This project demonstrates a simple Python project structure using VS Code.  
It includes utility functions, a sample API call, and Git/GitHub workflow practice.

## Project Structure

ai-api-vscode-assignment
│
├── src
│   ├── main.py
│   ├── api_client.py
│   └── utils.py
│
├── requirements.txt
├── notes.txt
└── README.md


## Features
- Demonstrates Python file organization
- Includes a simple API client example
- Uses a public test API
- Shows Git and GitHub workflow

## How to Run

Install dependencies:


pip install -r requirements.txt


Run the project:


cd src
python main.py
